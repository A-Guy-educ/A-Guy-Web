# Shared Components

Project-wide reusable components that can be used across the entire application.

## Directory Structure

- **Typography/** - Text and heading components
- **Layout/** - Layout primitives (Stack, Grid, Section)
- **Icon/** - Icon wrapper components
- **Loading/** - Loading states (Skeleton, Spinner)
- **EmptyState/** - Empty and error state components

## Guidelines

- All components must use 100% Tailwind CSS
- Use design tokens from `tailwind.tokens.mjs`
- Include TypeScript types
- Add comprehensive Storybook stories
- Support dark mode
- Follow accessibility best practices
