### Scenario

User Story
As a student, I want the option to select exam topics directly from my registered course lessons—instead of only typing them manually—so that my study plan is more accurate and provides direct access to the relevant course content.

Acceptance Criteria
Hybrid Input System: Maintain the existing free-text input for custom topics, but add a primary "Select from Course" toggle/section.

Course Lesson Integration: Automatically pull and display the syllabus (lessons, modules, and sub-topics) for the course the user is currently preparing for.

Selection-to-Plan Sync: When a user selects specific lessons from the list, they are automatically added to the material list without requiring manual typing.

Smart Navigation Buttons: The study plan generated from these selections must include buttons/links that navigate the user directly to the corresponding lesson page or video player within the course.

Dynamic Content Mapping: If an instructor updates a lesson title or content, the link in the student's study plan must remain valid or update accordingly.
