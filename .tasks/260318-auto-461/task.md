# Task

## Issue Title

Lesson introduction page is not editable and does not support formatting or block-based content
# 🐞 Bug Report

## 1. Title
Lesson introduction page is not editable and does not support formatting or block-based content

## 2. Environment
- Environment: prod
- Browser / Device: Chrome
- User Role / Tenant: Student

## 3. Preconditions
The user has access to the Admin panel.

At least one lesson exists in the system.

The lesson includes an introduction/explanation section.

The user has permissions to edit lesson content.

The system supports the block editor in other areas (for comparison).

## 4. Steps to Reproduce
Go to the Admin panel.

Open an existing lesson or create a new one.

Navigate to the lesson introduction/explanation section.

Try to add or edit content using formatting options (e.g., layout, media, structured content).

Save the lesson.

Open the lesson as a user.

## 5. Expected Result
The introduction page should be fully editable using the block system

Admin should be able to design the page using:

text blocks

images

videos

graphs

any other existing block types

The page should support full formatting and layout control

## 6. Actual Result
The lesson introduction/explanation page cannot be properly designed or formatted.

Currently:

The content appears as plain/unstructured text

No formatting tools are available

The page does not support the block system used in the admin

As a result, the introduction page looks limited, inconsistent, and difficult to design, especially compared to other parts of the system that use structured blocks.

## 7. Priority
🟠 P1 — High

## 8. Reproducibility
always
