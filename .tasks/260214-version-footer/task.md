# Version Footer Task

## Goal

Display the version number from `package.json` in the site footer.

## Requirements

- Read version from `package.json`
- Show it in the public footer (minimal styling, subtle)
- Commit and open PR

## Implementation

1. Read `package.json` to get the version
2. Add version display to the footer component
3. Commit to branch `feature/version-footer`
4. Open PR

## Deliverable

PR with the version footer implemented.
