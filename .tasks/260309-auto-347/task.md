# Task

## Issue Title

The video is not displayed to the user
# 🐞 Bug Report

## 1. Title
The video is not displayed to the user

## 2. Environment
- Environment: prod
- Browser / Device: Chrome
- User Role / Tenant: Student

## 3. Preconditions
When uploading a video in the admin panel, it is saved successfully but is not displayed to the student on the frontend.

## 4. Steps to Reproduce
Go to the Admin panel.

Open or create a lesson/exercise.

Add a video to a block inside the exercise (upload or attach a video).

Save the changes.

Open the exercise as a student / frontend view.

## 5. Expected Result
The uploaded video should be displayed to the student in the exercise block and on the lesson introduction

## 6. Actual Result
The video is saved in the admin panel but is not displayed to the user.

## 7. Reproducibility
always
