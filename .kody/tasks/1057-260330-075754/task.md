### Scenario

User Story
As a user, I want the teacher selection interface and profiles to be presented in my preferred language (Hebrew or English) so that I can navigate the selection process comfortably regardless of which version of the website I am using.

Acceptance Criteria
Dynamic UI Translation: The system must detect the user's current site version. If the user is on the Hebrew version, all UI strings must be displayed in Hebrew. If on the English version, show English equivalents.

Profile Content Localization:

Implement dual-language fields for teacher profiles (Bio, Specializations, and Titles).

The system must serve the content version that matches the user's active language setting.
