# Task

## Issue Title

Lesson chat starts from the first message instead of the latest message when entering the lesson
# 🐞 Bug Report

## 1. Title
Lesson chat starts from the first message instead of the latest message when entering the lesson

## 2. Environment
- Environment: prod
- Browser / Device: Chrome

## 3. Preconditions
The user is logged into the platform.

The lesson contains a chat interaction with multiple messages.

The user enters or resumes the lesson.

## 4. Steps to Reproduce
Open a lesson that contains a chat with multiple messages.

Exit the lesson or refresh the page.

Enter the lesson again.

## 5. Expected Result
The chat should automatically scroll to and display the latest message when the lesson opens.

## 6. Actual Result
The chat opens at the first message in the conversation.

## 7. Reproducibility
always
