# About A-Guy

A-Guy is an **AI-native educational platform** — built from the ground up as
an "AI Operating System for Education," not a typical website or a classic LMS.

Where most platforms bolt AI on as a chatbot or recommendation widget, A-Guy
makes AI part of the core architecture: a context-aware tutor that lives inside
the learning flow, long-term memory that persists across sessions, and semantic
(vector) retrieval over the content itself. The first domain is deep
mathematics learning, delivered through a structured course hierarchy and an
embedded tutor rather than linear page-by-page navigation.

It is a single system that unifies a learning platform, a general content
management system (pages, prompts, configuration), an intelligent chat/tutor
layer, and multi-tenant + multi-locale infrastructure. Content and UI are
bilingual (Hebrew/English).

In one line: A-Guy manages **knowledge, context, and action** — not just content.
