# Task

## Issue Title

Document/PDF not displayed on Desktop Chrome but visible on mobile for the same user
# 🐞 Bug Report

## 1. Title
Document/PDF not displayed on Desktop Chrome but visible on mobile for the same user

## 2. Environment
- Environment: prod
- Browser / Device: Chrome

## 3. Preconditions
The user is logged into the platform.

A lesson/screen contains an uploaded PDF/document.

The file was uploaded successfully in the Admin panel.

The document preview works in admin.

## 4. Steps to Reproduce
Log in to the platform as a user.

Open the lesson/screen that contains the uploaded document.

View the document on Desktop Chrome.

## 5. Expected Result
The document should display correctly on Desktop Chrome, just as it does on mobile.

## 6. Actual Result
The document viewer appears blank on Desktop Chrome.

The document content is not displayed.

## 7. Reproducibility
always
