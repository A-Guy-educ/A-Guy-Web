## Issue Title

Lesson introduction container is too narrow on both desktop and mobile causing content display issues
# 🐞 Bug Report

## 1. Title
Lesson introduction container is too narrow on both desktop and mobile causing content display issues

## 2. Environment
- Environment: preview
- Browser / Device: Chrome
- User Role / Tenant: Student

## 3. Preconditions
The user is logged into the platform.

A learning lesson exists in the system.

The lesson includes an interactive lesson.

The interactive lesson contains introduction content.

## 4. Steps to Reproduce
Open a learning lesson in the system.

Select an interactive lesson.

Click Start Lesson.

View the lesson introduction section.

## 5. Expected Result
The introduction section should adapt properly to screen size.

Content should be displayed in a readable layout without unnecessary narrow spacing.

Desktop: increase container width to approximately 1200–1280px max-width.

Mobile: allow the container to use full available width with proper padding (16–20px).

## 6. Actual Result
The introduction container is too narrow on desktop and mobile.

Content does not utilize the available screen space properly.

In some cases, the layout appears broken or difficult to read.

## 7. Reproducibility
always
