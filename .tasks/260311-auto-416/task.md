# Task

## Issue Title

The lesson order defined in the admin panel is not reflected on the website
# 🐞 Bug Report

## 1. Title
The lesson order defined in the admin panel is not reflected on the website

## 2. Environment
- Environment: prod
- Browser / Device: Chrome

## 3. Preconditions
The user is logged into the Admin panel.

Lessons exist in the system.

Each lesson has a defined order/number in the admin.

## 4. Steps to Reproduce
Go to the Admin panel.

Open the lessons management section.

Set a specific order/number for the lessons.

Save the changes.

Open the lessons section on the website (for example: exam 471).

Observe the lesson order.

## 5. Expected Result
Lessons should be displayed according to the order defined in the admin panel.

## 6. Actual Result
The lessons on the website are not displayed according to the defined order in the admin.

## 7. Reproducibility
always
