# Site Documentation

This directory contains scenario-first development artifacts.

## Structure

```
site-docs/
├── prototypes/     # HTML prototypes from designers
├── behaviors/      # Site behavior JSON specifications
└── prds/          # Generated Product Requirements Documents
```

## Usage

### Prototypes

Place HTML prototype files here. These are reference designs - the Design System is the source of truth.

### Behaviors

JSON files documenting how the site handles various situations:

- Loading states
- Error handling
- Authentication flows
- Animations
- Responsive behavior

### PRDs

Generated from scenarios during the architect stage.
