# Kody memory index

One line per memory. The `memory-writer` job maintains this file — do not edit by hand.
To add a memory, drop a sticky note in [`inbox/`](inbox/README.md) and the job will file it.

Each entry: `- [Title](id.md) — one-line hook (type: <type>)`.

Memory is sacred — it only grows from explicit operator action: saying "remember
X" in chat, tagging a dashboard message with `#remember`/`#lesson`/`#decision`,
or clicking Approve/Reject/Dismiss on a CTO recommendation. Auto-capture of
git activity belongs in `git log`, not here.
